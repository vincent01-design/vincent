My Recent Project

I and my team we developed a simple Web Registration and Search System with Firebase, Google Maps API, and Webhooks

This Web Registration and Search System is a simple platform that lets users register their information and search for it when needed. Using HTML, CSS, and a little JavaScript for interactivity, it uses Firebase to securely store data, Google Maps API to add location-based search, and Webhooks to ensure real-time updates. Users can enter their details, search for specific records, and view results on an interactive map. This system is fast, easy to use, and mobile-friendly, making it perfect for things like service directories, event sign-ups, or any platform that needs to search and display location-based data.

My Calculator

I created a simple calculator using Visual Studio and C#. The calculator allows users to perform basic operations like addition, subtraction, multiplication, and division. To build it, I used Windows Forms to design the user interface (UI), adding buttons for numbers and operations. Then, I wrote C# code to handle the logic behind the calculations, making sure the calculator responds to user input and performs the correct operation. The result is displayed in a text box, and the app is easy to use, with a clean and simple design.
