First Coding Project

About This Project

My first coding project was a Student Information System, which I built with a partner and the language that we decided to use is C#. The system manages and stores student data, including names, IDs, grades, department, and course information. It allows users to add, update, and view student records through a simple interface. Working on this project together was a great experience, as it taught us how to organize data and build a functional, user-friendly system. It also gave us a solid foundation in coding, which has been helpful for the projects we've worked on since.

Technical Details

For my first project, my partner and I built a Student Information System using C#. The system is designed to manage student data, such as names, IDs, department, and course details. We used C# for the coding and Windows Forms to create a simple, user-friendly interface with buttons, text boxes, and tables to display and manage the records. All the student information is stored in a Microsoft SQL Server database, making it easy to save, update, and access the data. The system lets users search and filter students by various criteria and sort records by name, ID, department, or course. We also made sure that the student IDs are unique, and the process of adding or deleting records is triggered by a click of a button. While we kept the design simple, the system runs smoothly with basic error handling.

Challenges Faced

One of the challenges we faced while working on the Student Information System was that we didn't have a laptop to use. Luckily, one of our classmates was really kind and let us borrow her laptop, which helped a lot. Another big challenge was that we had no idea how to even start the project because it was our first time making a system. We were both new to this kind of work, so it was really difficult and challenging at first. But with time and a lot of effort, we figured things out, and the experience ended up being a great learning opportunity.
